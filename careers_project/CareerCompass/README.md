# CareerCompass

A beginner-level Python course project that helps a student explore careers, match jobs, and track applications.

## Project Structure

- `main.py` - App entry point using `tkinter`.
- `database/` - Database package with SQLite setup and helper functions.
- `gui/` - GUI pages for login, dashboard, quiz, jobs, and application history.
- `services/` - Simple recommendation and job-matching logic.

## How to run

1. Open the `CareerCompass` folder in your editor.
2. Run the app with:

   ```bash
   python main.py
   ```

3. Follow the login prompt, then use the dashboard buttons to explore the app.

## Notes

- The app stores data in `career_compass.db`.
- It is written in a simple, beginner-friendly style with comments and plain logic.
